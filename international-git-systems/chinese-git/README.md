# Chinese Git Systems Integration

This directory contains configurations for Chinese Git hosting services.

## Supported Systems

- Gitee (Chinese GitHub alternative)
- GitLab China
- Local Chinese providers

## Setup

1. Host on Chinese Git service for Great Firewall compliance.
2. Configure CI/CD with Chinese data centers.
3. Support for Chinese localization and regulations.

## Pipelines

- Builds on Chinese cloud providers (e.g., Alibaba Cloud).
- Compliance with Chinese cybersecurity laws.
