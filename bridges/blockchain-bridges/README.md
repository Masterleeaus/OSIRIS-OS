# Blockchain Bridges

This directory contains bridges to various blockchain networks.

## Supported Blockchains

- Ethereum
- Polkadot
- Binance Smart Chain
- Solana
- And more...

## Setup

1. Install necessary SDKs (e.g., web3.js, polkadot.js).
2. Configure RPC endpoints and API keys.
3. Set up wallets and signers.

## Examples

- Polkadot Bridge: Connect to Polkadot parachains.
- Ethereum Bridge: Interact with ERC-20 tokens.
