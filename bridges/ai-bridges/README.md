# AI Bridges

This directory contains bridges to external AI models and services.

## Supported AI Systems

- OpenAI GPT models
- Generative AI APIs
- Custom AI models
- MCP (Model Context Protocol) integrations

## Setup

1. Obtain API keys for services (e.g., OpenAI API key).
2. Install SDKs: `npm install openai`
3. Configure endpoints.

## Examples

- GPT Integration: Use for chat and completion.
- Gen AI Bridge: For image or text generation.
