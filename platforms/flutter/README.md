# Flutter Platform

This directory contains the Flutter cross-platform implementation for AIPlatform.

## Setup

1. Install Flutter SDK and Android Studio or Xcode.
2. Install dependencies: `flutter pub get`.
3. Configure build settings in `flutter/pubspec.yaml`.
4. Run `npm run build:flutter` to build the app.

## Features

- Cross-platform UI (iOS, Android, Web, Desktop).
- Web5 DID integration via plugins.
- AI model inference on device.
