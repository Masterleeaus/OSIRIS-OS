# Aurora Platform

This directory contains the Aurora-specific implementation for AIPlatform.

## Setup

1. Install Aurora development tools and SDK.
2. Install dependencies: `npm install`.
3. Configure build settings in `aurora/config.xml`.
4. Run `npm run build:aurora` to build the app.

## Features

- Native Aurora UI components.
- Integration with Aurora ecosystem for AI and blockchain.
- Custom platform optimizations.

## Documentation

- [Aurora Setup Guide](./docs/setup.md)

For development:
```bash
npm run dev:aurora
```

For production:
```bash
npm run build:aurora
```
