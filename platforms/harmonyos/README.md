# HarmonyOS Platform

This directory contains the HarmonyOS-specific implementation for AIPlatform.

## Setup

1. Install HarmonyOS DevEco Studio.
2. Install dependencies: `npm install`.
3. Configure build settings in `harmonyos/config.json`.
4. Run `npm run build:harmonyos` to build the app.

## Features

- Native HarmonyOS UI components (ArkUI).
- Integration with Huawei services.
- Support for distributed apps.

## Documentation

- [HarmonyOS Setup Guide](./docs/setup.md)
- [API Reference](./docs/api.md)

## Building

For development:
```bash
npm run dev:harmonyos
```

For production:
```bash
npm run build:harmonyos
```
