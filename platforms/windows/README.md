# Windows Platform

This directory contains the Windows-specific implementation for AIPlatform.

## Setup

1. Install Visual Studio and Windows development tools.
2. Install dependencies: `npm install`.
3. Configure build settings in `windows/app.manifest`.
4. Run `npm run build:windows` to build the app.

## Features

- Native Windows UI components.
- Support for Web3 blockchain integrations.
- AI training and inference on Windows hardware.

## Documentation

- [Windows Setup Guide](./docs/setup.md)

## Building

For development:
```bash
npm run dev:windows
```

For production:
```bash
npm run build:windows
```
