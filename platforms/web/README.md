# Web Platform

This directory contains the Web-specific implementation for AIPlatform.

## Setup

1. Install Node.js and npm.
2. Install dependencies: `npm install`.
3. Configure build settings in `web/vite.config.js` or similar.
4. Run `npm run dev` to start development server.

## Features

- Responsive web UI with React/Vue.
- Web5 integration for decentralized web.
- PWA (Progressive Web App) support for offline use.

## Building

For development:
```bash
npm run dev
```

For production:
```bash
npm run build
```
