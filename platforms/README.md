# Platforms

This directory contains platform-specific implementations and configurations for AIPlatform.

## Available Platforms

- **iOS**: Mobile iOS application setup, build configs, and native integrations.
- **macOS**: Desktop macOS application, including Electron or native builds.
- **Windows**: Windows desktop and UWP applications.
- **Linux**: Linux desktop applications, including AppImage and Snap builds.
- **Web**: Web-based application, optimized for browsers.
- **WinUWP**: Universal Windows Platform for Windows Store.
- **Aurora**: Custom or experimental platform integrations.
- **Android**: Mobile Android application using React Native.
- **Tizen**: Tizen OS for Samsung devices and IoT.
- **HarmonyOS**: Huawei's HarmonyOS for distributed apps.
- **Flutter**: Cross-platform application for mobile, web, and desktop.
- **tvOS**: Apple TV application.

Each platform folder includes:
- Build configurations (e.g., package.json, build scripts).
- Platform-specific code modules.
- Documentation and setup guides.

## Getting Started

Navigate to the specific platform folder and follow the README there for setup instructions.
