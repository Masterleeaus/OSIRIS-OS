# Verticals

This directory contains vertical slices for domain-specific features in AIPlatform.

## Purpose

Verticals represent domain-specific modules or features, such as AI training, blockchain governance, or user management.

## Structure

- `ai-training/`: Vertical for AI model training and management.
- `blockchain-governance/`: Vertical for DAO and voting mechanisms.
- `data-marketplace/`: Vertical for data and model marketplaces.
- `user-identity/`: Vertical for user identity and Web5 integrations.

## Implementation

Each vertical is a self-contained module with its own controllers, models, and views.
