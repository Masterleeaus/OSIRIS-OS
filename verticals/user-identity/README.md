# User Identity Vertical

This vertical manages user identities, DIDs, and Web5 integrations.

## Features

- Decentralized identity management
- Verifiable credentials
- DID creation and management
- Identity verification

## Setup

1. Install Web5 SDK
2. Configure DID methods

## API Endpoints

- POST /api/identity/did - Create DID
- GET /api/identity/credentials - Get credentials
- PUT /api/identity/verify - Verify identity
