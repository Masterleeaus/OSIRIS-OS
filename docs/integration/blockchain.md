# Blockchain Integrations Documentation

## Overview

AIPlatform integrates with various blockchains for decentralized features.

## Supported Blockchains

- **Polkadot**: For cross-chain interoperability.
- **Ethereum**: For smart contracts and tokens.

## Setup

1. Install dependencies: `npm install @polkadot/api ethers`
2. Configure RPC endpoints in config.
3. Set up wallets for signing.

## Usage

- Use bridges in `bridges/blockchain-bridges/` for interactions.
- Query chain data, send transactions.

## Security

- Use secure RPC providers.
- Validate transactions before signing.

## Troubleshooting

- Check network connectivity.
- Verify API keys and endpoints.
