# macOS Platform Documentation

## Overview

Desktop macOS app using Electron.

## Setup

1. Install Node.js 20+.
2. Run `npm install`.
3. Run `npm run dev:macos` for development.

## Features

- Native macOS window
- Menu bar integration
- File system access

## Building

- Use `npm run build:macos` for production.
- Package with electron-builder.

## Deployment

- Distribute via DMG or App Store.

## Troubleshooting

- Ensure Electron is installed.
- Check for macOS-specific permissions.
