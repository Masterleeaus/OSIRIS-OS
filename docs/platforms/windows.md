# Windows Platform Documentation

## Overview

Desktop Windows app using Electron.

## Setup

1. Install Node.js 20+.
2. Run `npm install`.
3. Run `npm run dev:windows`.

## Features

- Native Windows UI
- Taskbar integration
- Windows APIs

## Building

- Use `npm run build:windows`.
- Package with NSIS.

## Deployment

- Distribute via EXE installer.

## Troubleshooting

- Ensure Windows development tools are installed.
- Check for Windows-specific errors.
