# Web Platform Documentation

## Overview

The Web platform provides a browser-based interface for AIPlatform.

## Setup

1. Ensure Node.js 20+ is installed.
2. Run `npm install`.
3. Run `npm run dev` for development.
4. Access at http://localhost:3000.

## Features

- Responsive design
- PWA support
- Web5 integration for decentralized identity

## API

- Web APIs for platform features
- REST endpoints for backend communication

## Troubleshooting

- Check console for errors.
- Ensure Vite is running.

## Deployment

Build with `npm run build` and serve from dist/.
