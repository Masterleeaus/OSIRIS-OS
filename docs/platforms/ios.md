# iOS Platform Documentation

## Overview

Native iOS app for AIPlatform.

## Setup

1. Install Xcode.
2. Run `npm install` for Capacitor.
3. Run `npx cap add ios`.
4. Run `npm run dev:ios`.

## Features

- Native iOS UI
- Camera and location access
- Web5 DID integration

## Building

- Use `npm run build:ios` for production build.
- Submit to App Store via Xcode.

## Troubleshooting

- Ensure iOS simulator or device is connected.
- Check Capacitor logs.

## Deployment

- Build with Xcode and upload to App Store.
